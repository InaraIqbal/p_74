Campus Life

School junior building


Built on around 2013

The junior building is a place of nostalgia for me. The junior bulding consists of classes from 1st to 4th grade. It has an auditorium, a music room, a computer lab and the nurse room.

Library


Built on around 2013

This is the library of the senior building. It has sections named aftre authors. Ruskin bond, Enid blyton, JK Rowling, Lily Singh etc... It has many reference books for projects and work that requires team work. Sometimes, we go to the library during the break times to complete home work/study or to read.

Assembly stage


Built on 2014

This is the assembly stage of our school. All grades use this stage. It has been very useful for us. We use this for assemblies, events like sports day, festival programmes etc.
